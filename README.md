# Scarvion App

Static landing page for Scarvion AI.

## Usage

```bash
npm install
```

Then open [public/index.html](public/index.html) directly in your browser.

No server or environment variables are required.

## Optional NPM Commands

```bash
npm start
```

Prints a reminder that this app is static.

## Deployment

Deploy the `public/` folder to any static hosting provider.
